
# Graph Search Algorithms

A Python implementation of Breadth-First Search (BFS) and A* Search using NetworkX on a weighted graph of Texas cities.

## Features
- Graph creation from CSV files
- Graph visualization
- Breadth-First Search (BFS)
- A* Search

## Setup
Install dependencies:

```bash
pip install -r requirements.txt
```

Place `cities.csv` and `distances.csv` in the `data/` folder before running the notebook.

This notebook is the original version submitted for the assignment and is preserved for portfolio purposes.
